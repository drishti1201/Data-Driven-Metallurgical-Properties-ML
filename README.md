# Data-Driven Analysis of Metallurgical Properties using Machine Learning

## Project Overview
This project applies Machine Learning techniques to analyze the relationship between alloy composition, heat treatment, microstructure, and mechanical properties.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Algorithms
- Exploratory Data Analysis (EDA)
- PCA
- K-Means Clustering
- Random Forest Regressor
- Gradient Boosting Regressor

## Dataset
- 150 samples
- 15 features
- Clean dataset with no missing values

## Results
- K-Means identified 3 meaningful material clusters.
- Random Forest achieved high prediction accuracy.
- Gradient Boosting slightly outperformed Random Forest.
- ML results validated metallurgical principles such as Hall-Petch relationship and martensite-hardness correlation.
